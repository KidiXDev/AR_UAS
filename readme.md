# Sunvale

A Unity project inspired by Stardew Valley.

## Features

- Farming mechanics
- Character interaction
- Day/night cycle

## Getting Started

1. Clone the repository.
2. Open the project in Unity (recommended version: 2021.3 or later).
3. Press Play to start exploring.

## Requirements

- Unity 6 (6000.34f1)
- .NET Framework 4.x
